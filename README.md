# Customer Support Ticket Triage System

## Project Description
This project analyzes customer support tickets to extract key insights, identify trends, and categorize issues. The goal is to help support managers prioritize tickets, optimize response times, and improve customer satisfaction. 

Key tasks performed:
- Cleaned and preprocessed ticket data
- Analyzed trends, priorities, and issue categories
- Visualized insights using charts
- Generated a Manager Report summarizing the findings

## Tools & Technologies Used
- **Python 3** – for data analysis and processing
- **Pandas** – for data manipulation and cleaning
- **Matplotlib & Seaborn** – for data visualization
- **Google Colab** – for interactive coding and notebook sharing
- **GitHub** – for version control and submission

## Files Included
- `Customer_Support_Tickets_Notebook.ipynb` – Colab notebook with analysis
- `customer_support_tickets_cleaned.csv` – cleaned dataset
- `Manager_Report_Customer_Support.pdf` – PDF report summarizing insights

## How to Run
1. Open the notebook in **Google Colab**.
2. Run all cells to reproduce the analysis and charts.
3. The cleaned CSV is generated and ready for use in any further analysis.
